# Noise Reduction and Signal Classification using Machine Learning

## Overview

This project focuses on noise reduction and signal classification using signal processing and machine learning techniques.

The system includes:

* Speech denoising using notch filtering
* ECG signal denoising
* Signal classification using Support Vector Machine (SVM)
* Feature extraction and analysis
* Visualization of results using plots and signal processing techniques

## Technologies Used

* Python
* Jupyter Notebook
* Machine Learning
* Scikit-learn
* NumPy
* Signal Processing
* SVM Classification

## Project Files

* final_denoiser_notebook.ipynb
* final_ecg_notch_project.ipynb
* signal_classification.py
* notch_example.py
* Trained SVM Models (.joblib)

## Features

* Noise reduction for speech signals
* ECG signal enhancement
* Signal classification pipeline
* Visualization of waveforms and processed outputs
